# 🧠 CNN on CIFAR-10

This project implements a Convolutional Neural Network (CNN) to classify images from the CIFAR-10 dataset.

## 📌 Features

* Image classification using CNN
* Built with TensorFlow / Keras
* Achieves good accuracy on test data

## 📂 Files

* `ashna_cnn_cifar.ipynb` → Main notebook

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/cnn-cifar10-project.git
```

2. Open the notebook

```bash
jupyter notebook
```

## 📊 Dataset

* CIFAR-10 dataset (10 classes of images)

## 🛠️ Tech Stack

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib
